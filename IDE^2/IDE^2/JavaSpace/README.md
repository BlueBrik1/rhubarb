# JavaSpace

JavaSpace is a Java editor GUI written in Rhubarbian. Its `.rhubarb` source uses `# RHUBARB_REVERSE_LINES`, so every non-comment code line is stored backwards.

Run it from the main `IDE^2` folder:

```bash
python3 rhubarb.py JavaSpace/java_editor.rhubarb
```

Open `JavaSpace/Hello.java`, then click **Compile** or **Run**.
